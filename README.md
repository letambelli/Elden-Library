# Elden Library

Este site foi desenvolvido como parte do projeto da disciplina de Programação Front-end, sendo uma apicação simples que simula um site de uma livraria usando HTML, CSS e javascript.


## Objetivo

O objetivo desse projeto consiste em desenvolver um sistema para cadastro de entidades, que podem ser fornecedores, clientes, escolas, ongs e etc.

No caso da Elden Library, o site usa HTML, CSS e javascript para apresentar uma página inicial e simular o cadastro de um usuário ou entidade.


## Tecnologias Utilizadas

- HTML
- CSS
- JS
